式(5-5) 从动锥齿轮转矩（发动机起动工况）：
Tce = Kd · k · Temax · i₁ · if · i₀ · η / n
Kd=动载系数；k=离合器接合系数；i₁=低档比；if=分动箱比；η=发动机到万向传动轴传动效率；n=驱动轴数

式(5-6) 从动锥齿轮转矩（驱动轮打滑工况）：
Tcs = G₂ · m₂' · φ · rr / (im · ηm)
G₂=驱动轴静载；m₂'=动轴质量系数；φ=附着系数；im=轮边减速比；ηm=主动轮到车轮效率

式(5-7) 从动锥齿轮转矩（日常行驶工况）：
Ft = mₐ · g · (fR + fH + fJ) Tcl = Ft · rr / (im · ηm · n)
fR=滚动阻力系数；fH=爬坡阻力系数（sinθ）；fJ=性能系数；rr=车轮滚动半径

主动锥齿轮（小齿轮）转矩：
Tc,主 = Tc,从 / (i₀ · ηG)  （强度用取 min(Tce,Tcs)；疲劳用取 Tcl）

⚡锥齿轮齿面力计算
齿宽中点处圆周力（式5-17）：
F = 2T / Dm2
T：从动齿轮转矩；Dm2：从动锥齿轮齿宽中点分度圆直径

从动齿轮齿宽中点分度圆直径（式5-18）：
Dm2 = D₂ − b₂·sinδ₂
D₂：从动锥齿轮大端分度圆直径；b₂：从动锥齿轮齿面宽；δ₂：从动锥齿轮节锥角

法向力分解（式5-19~5-21）：
F = FT·cosα·cosβ   FN = F·tanα / cosβ   Fs = F·tanβ

主动锥齿轮轴向力与径向力（式5-22~5-23）展开形式：
Fax = F · (tanα·sinγ ± sinβ·cosγ) / cosβ
FRx = F · (tanα·cosγ ∓ sinβ·sinγ) / cosβ
γ = 节锥角（主动轮取 δ₁，从动轮取 δ₂）；β = 中点螺旋角；F = 齿宽中点圆周力（F = 2T/Dm）
符号正负由螺旋方向与旋转方向组合决定，见下表；从动齿轮螺旋分量符号与主动齿轮相反。

📐桥壳截面参数（式5-57/58 扩展）
◆ 圆环截面（式5-57/58）
惯性矩:　Iz = Iy = π(D₁⁴ − d₁⁴)/64　（轴对称，两方向相同）
式5-57: 弯曲截面系数 Wz = Wy = π(D₁⁴ − d₁⁴)/(32D₁)　（mm³）
式5-58: 扭转截面系数 Wp = π(D₁⁴ − d₁⁴)/(16D₁)　（mm³）= 2Wz
　　　　（Wz=Wy，垂直/水平弯曲截面系数相同，无需区分方向）
◆ 矩形空心截面（材料力学 + Bredt薄壁闭合截面公式）
外宽 B_r × 外高 H_r，内宽 b = B_r−2t_b，内高 h = H_r−2t_h

垂直弯曲（抵抗垂直载荷弯矩，绕水平z轴）：
　Iz = (B_r × H_r³ − b × h³)/12　（mm⁴）
　Wz = Iz/(H_r/2) = (B_r × H_r³ − b × h³)/(6 × H_r)　（mm³）

水平弯曲（抵抗侧向载荷弯矩，绕垂直y轴）：
　Iy = (H_r × B_r³ − h × b³)/12　（mm⁴）
　Wy = Iy/(B_r/2) = (H_r × B_r³ − h × b³)/(6 × B_r)　（mm³）

扭转（Bredt薄壁闭合截面公式）：
　Am = ((B_r+b)/2) × ((H_r+h)/2)　（中线围合面积，mm²）
　t_min = min(t_h, t_b)　（最小壁厚）
　τ_max = T/(2 × Am × t_min)　→　Wp = 2 × Am × t_min　（mm³）

注：当 Wz≠Wy 时，工况二侧滑须用 Wy 校核水平面弯曲应力！
🔢三工况桥壳强度校核（式5-56/57/58）
工况一（式5-56）牵引力或制动力最大时：
　Fz2 = m₂ʹG₂/2（垂直反力），Fx2 = Fz2·φ（牵引/制动力）
　Mv = Fz2·b（垂直弯矩），Mh = Fx2·b（水平弯矩）
　Tт = Fx2·rr（扭矩）
　σ = Mv/Wv + Mh/Wh（垂直+水平弯曲应力叠加）
　τ = Tт/Wт（扭转切应力）
　σn = √(σ²+4τ²)（第四强度理论合成应力）

工况二（式5-57）侧向力最大时（侧滑，φ₁=1.0）：
　Fz₂ₒ = G₂(0.5+hg·φ₁/B₂)（外侧车轮垂直反力）
　Fz₂ᵢ = G₂−Fz₂ₒ（内侧车轮垂直反力）
　σ₁ = Fz₂ₒ·(b + φ₁·rr) / Wv（外侧板簧座，应力较大）
　σ₂ = Fz₂ᵢ·(b − φ₁·rr) / Wv（内侧板簧座，应力较小）
　（侧向摩擦力效果已通过 b±φ₁·rr 等效力臂纳入，无需另算水平弯曲）

工况三（式5-58）汽车通过不平路面时：
　σ = k·G₂·b / (2·Wv)（纯垂直弯曲）

式中 b = 板簧座中心到轮毂中心距离（mm）；Wv=垂直抗弯截面系数；
Wh=水平抗弯截面系数；Wт=抗扭截面系数。
许用值：[σ]=300~500 MPa，[τ]=150~400 MPa（铸铁取小值，钢板冲压焊接式取大值）
